Cours 3

Utilisation de boucles : while, do while, for

Tâche à réaliser

Difficulté : facile

Créez un script contenant une boucle while qui permet à l'utilisateur de donner un nombre plus grand que 10, et tant que celui-ci n'est pas plus grand que 10 redemandez un nombre. Affichez ce nombre dans une balise html.

Explications supplémentaires

Étant donné qu'on va demander à l'utilisateur un nombre on utilisera la commande prompt pour enregistrer la valeur dans une variable qu'on nommera "number". Ensuite on utilisera une boucle "while" qui testera à chaque fois si "number" dépasse 10 ou non. Si elle est toujours inférieure à 10, on redemande à l'utilisateur un nombre. Si elle (la variable) est supérieure à 10 alors inscrivez le nombre donné par l'utilisateur dans une balise hmtl ciblée par son ID.

Les ressources pour y parvenir

  • Cibler un élément html par son id
  • Utilisation de la "fonction" prompt en js
  • Utilisation de while en js
  • Remplacer le contenu d'une balise html par un autre en js

Tâche à réaliser

Difficulté : facile-intermédiaire

Créez un script utilisant une boucle while contenant une incrémentation. La console du navigateur doit afficher tous les nombres entre 20 et 30.

Explications supplémentaires

Même début que l'exercice précédent, on crée une variable contenant un nombre qui servira pour l'incrémentation. On crée une boucle while avec une condition permettant d'arrêter la fonction lorsqu'on dépasse 30.

Les ressources pour y parvenir

  • Incrémenter une variable
  • Utilisation de while en js

Tâche à réaliser

Difficulté : intermédiaire-difficile

Créez un script utilisant une boucle for, contenant une incrémentation qui permet de demander à l'utilisateur des prénoms. Lorsque l'utilisateur n'a plus de prénom à donner, il n'inscrit rien dans l'invite de commande et appuie sur validé. Une fois tout cela réalisé, affichez dans une alert les prénoms donnés par l'utilisateur ainsi que leur nombre.

Explications supplémentaires

On crée une boucle for avec une incrémentation et une variable qui contiendra les prénoms. On utilise l'incrémentation pour compter le nombre de prénoms utilisés. Utilisez une condition qui vérifie si la variable utilisée pour les prénoms est bien remplie, si oui rajoutez les prénoms précédents au dernier introduit. Si non, arrêtez la fonction et affichez dans une fonction "alert" le nombre de prénoms (avec la variable incrémentée) ainsi que les prénoms (grâce à la variable qui a enregistré les prénoms).
