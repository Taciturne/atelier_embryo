Partie 2

Cibler un élément html par son ID, utiliser une variable booléenne pour changer sa couleur

un carré tout beau

Tâche à réaliser

Lorsqu'on clique sur le bouton, la couleur du carré doit changer. Il (le carré) doit pouvoir changer de couleur entre les deux à chaque click.

Explications supplémentaires

Nous disposons d'un bouton et d'un carré. Pour le moment, lorsqu'on clique sur le bouton il ne se passe rien. Le but est donc, en javascript, de réussir à "écouter" l'événement du click sur le bouton en premier lieu. Une fois cela réussi, vous devez réussir à cibler le carré en js par son ID (de la même manière que le bouton) et lui donner la condition suivante : lorsque la variable booléenne est vrai, rajouter la class .color_green et retirer la class .color_blue. Inversément, lorsque la variable booléenne est fausse, rajouter la class .color_blue et retirer la class .color_green.
